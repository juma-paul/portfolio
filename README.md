# Portfolio

Minimal static portfolio site.

**Stack:** HTML, CSS, JavaScript
**Hosting:** GitHub Pages

## Run Locally

```bash
npx serve
```

## Edit Content

Update `data.js` to change skills, projects, certifications, and blog posts.

## Deploy

Push to `main` → auto-deploys.

**Live:** [juma-paul.github.io/portfolio](https://juma-paul.github.io/portfolio/)
